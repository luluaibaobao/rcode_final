filter.R:
Function to filter the price and signal curve and put them along with the raws into a dataframe.

turns_filter.R:
Functions to find and plot turn points and swing points.

cal_target.R:
Function to change the turning points into target position.

cal_pos.R:
Function to change the target position into actual position.

pnl.R:
Function to calculate the return.

pos_csv.R:
Function to put each actual trade information into a csv file.
